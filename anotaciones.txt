Crear una aplicación web en Streamlit para predecir la calidad del vino utilizando modelos de Machine Learning entrenados con sklearn.
La aplicación debe:
- cargar modelos entrenados en formato .pkl usando joblib
- permitir ingresar variables fisicoquímicas del vino
- realizar predicciones de calidad del vino
- mostrar si el vino es de buena o mala calidad
- incluir interfaz interactiva con Streamlit
- incluir nombre del estudiante
- incluir código ISIL
- incluir enlace al cuaderno de Google Colab
- utilizar Random Forest como modelo principal
